----

**O objetivo deste documento é validar e homologar o processo para retirar Pedidos do Faturamento Automático.**

----




**1. Para iniciar o processo, vamos acessar o seguinte caminho...   
Modulo: 97 – Distribuição de Peças  
Menu: Atualizações -> Documentos -> Nfe Sefaz   
Botão: Outras Ações -> Retirar Ped Fat**


**2.  Após clicar no botão ‘Retirar Ped Fat’, o usuário deverá informar o número do pedido que deseja retirar do faturamento, a rotina só aceitará pedidos que esteja com status faturamento Automático.**
