### Especificação da customização

Este ajuste tem como objetivo importar lista de preço

----

### Especificação de funções e rotinas

* **CM010MNU** - Rotina onde será incluido a customização

----

### Especificação de parametros

Nenhuma

### Execução do Processo

* Acesso a rotina
Acessar Modulo Maquinas 97
Atualizações => Compras => ADM Compras => Tabela de Preços

Vá em outras ações => Importa Lista
![fd](img\TabPrc01.jpg)

Ao clicar na rotina, vai aparecer a tela para informar os dados do Fornecedor, prencha os dados e clique em **[OK]**
![fd](img\TabPrc02.jpg)

Vai abrir uma janela para selecionar o arquivo a ser importado, o arquivo precisa estar com as colunas conforme imagem abaixo.

![fd](img\TabPrc03.jpg)

Após o processo acima a lista vai ser importada.

Faça a importação da lista utilizando o mesmo forncedor, após informar os dados do fornecedor, vai apresentar a mensagem que o fornecedor já possui uma lista e perguntar se deseja reimportar caso queira atualizar a lista, clique em  **[SIM]**
![fd](img\TabPrc04.jpg)
